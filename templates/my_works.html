A Collection Of My Work

The collection below is of some projects I have built over my time studying to become a python developer, as well as projects that I have created on my own. I have included screenshots and short descriptions to give a basic idea of my capabilities.

The classic Snake game that can keep track of all time high scores.

A password manager that allows you to save passwords to a text file as well as to generate and search for passwords that have been created and saved.

An Etch a Sketch game that allows you to emulate the classic Etch a Sketch boards we all had as kids.

A flash card program to help you study spanish words. It contains a right or wrong button so that you can revisit the ones you do not know.

A habit tracker to track certain activities done in a day and for how long. This data will then be uploaded to Pixela to provide visual feedback.

A dummy website for a moving company called Move With Joy. Github link: https://github.com/DiscoKitti/move_with_joy_website.git

An app for common knowledge quiz questions. It can keep track of score, and will show you whether your answer is correct or not.

A website based Tinder for dogs. Github link: https://github.com/DiscoKitti/tindog_website.git

A website that contains some of my top movies of all time. Github link: https://github.com/DiscoKitti/top_10_movies_website.git
